## üìù SEO Checklist for New Content

### Before Publishing Any New Page:
- [ ] Add unique, descriptive page title (50-60 characters)
- [ ] Write compelling meta description (150-160 characters)
- [ ] Include 3-5 relevant keywords naturally
- [ ] Add SEO component with proper props
- [ ] Use H1 tag (only ONE per page)
- [ ] Use H2, H3 for subheadings in order
- [ ] Add alt text to ALL images
- [ ] Include internal links to other pages
- [ ] Update sitemap.xml if needed

### Image SEO Best Practices:
```html
<!-- Good Example -->
<img 
  src="course-image.jpg" 
  alt="IELTS coaching classroom at Global SR Academy Kovai Pudur, Coimbatore with students and instructor"
  title="IELTS Preparation Course"
  loading="lazy"
/>

<!-- Bad Example -->
<img src="image1.jpg" alt="image" />
```

### Internal Linking Strategy:
- Link from homepage to all major pages
- Link from course pages to founder page (builds authority)
- Link from blog posts to relevant courses
- Use descriptive anchor text (not "click here")
